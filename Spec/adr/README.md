# Architecture Decision Records (Web Transition)

This folder contains accepted architecture decisions for the web app migration.

## Accepted ADRs

1. `ADR-001-board-library.md`
2. `ADR-002-stockfish-distribution.md`
3. `ADR-003-browser-storage-schema.md`
4. `ADR-004-mobile-performance-guardrails.md`

## Usage

1. Treat accepted ADRs as binding unless superseded by a new ADR.
2. When implementation conflicts with an ADR, either:
   - align code with ADR, or
   - write a replacement ADR and mark old one superseded.
