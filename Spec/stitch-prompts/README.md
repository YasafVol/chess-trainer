# Stitch Prompt Pack

These files are copy-paste prompts for Google Stitch.

Use them in this order:

1. `landing_prompt.md`
2. `workbench_prompt.md`
3. `puzzles_prompt.md`

Recommended process:

1. Paste the base prompt for one surface into Stitch.
2. Generate 2 to 4 variants.
3. Pick one winner.
4. Use the refinement prompts in the same file to push that winner.
5. Export the chosen screen to Figma.
